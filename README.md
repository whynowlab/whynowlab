## Architecture: Jarvis AI OS

```
Jarvis (AI OS)        — policy engine, model routing, 4-tier memory
  ↓
Orchestrator          — 22 teams, fan-out-fan-in, quality gates
  ↓
Skills + Hooks        — 329 skill mappings, 11 event hooks
  ↓
Claude Code CLI       — iTerm2 + tmux + real-time dashboard
```

> 4개 AI 엔진 (Claude, Gemini, GPT-5.4, Codex)을 목적에 맞게 라우팅.
> Claude Code가 시스템의 90%를 스스로 구축했습니다.
